Fonts

Choosing right font and right style is a subtle way of guiding the user to the right emotion and understanding of your website.

Headings

Be sure to use headings correctly. They are used for hierarchy, NOT for styling purposes.

The font used for headings is Playfair Display from Google Fonts. This is a serif font.

Example of Heading One

This is used on the main heading. Only one per page. Size: 3rem.

Example of Heading Two

This is used for sub-headings of h1. Size: 2rem.

Example of Heading Three

This is used for sub-headings of h2. Size: 1.5rem.

Example of Heading Four

This is used for sub-headings of h3. Size: 1.3rem.

Body Text and Button Text

The font used for body text and button text is Fauna One. This is also a serif font, and it goes well with the font for the headings. This one is more playful and fits the theme of the site. Size for body text: 1rem.

This is an example of body text, with Fauna One font and correct color, which is primary color, #024274.

Body text should always be in this color, unless there is another background than the standard body background.

If light background and not use of primary color, the body color should be black, #000.

If there is a dark background, body color should always be white, #fff.
