Components

Here are different website components.

Logo

Logo is what makes a company stand out visually against competitors. The logo and correct use of logo, is very important in brand building and promotion.

Specifications on use

Placement:

Top left corner on navigation bar, desktop size. Centered on ipad/mobile size.

Margin:

There should be enough space around the logo, minimum 1em on each side. Best practise is 1em on the sides, 2em top and bottom.

Size:

200px in navigation menus. If bigger is needed: max 250px. Smallest size: 150px.

Colors:

The main logo is made in two versions, depending on the background - white text or blue text. It also comes in a black version for when that is needed. No other color combinations allowed.

Different Color Combinations:

White Text on Primary Medium Background

When to use:

This is used when there is a colored background. Preferably primary medium as background color.

White Text on Black Background

When to use:

When there is need for having black/white color scheme only.

Black Text on White Background

When to use:

When need of black/white color scheme.

Interactive Features

Graphics and Icons

Graphics and Icons used on the webpage should accompany text and not be used only for decoration.

Icons and Social Media Icons should all follow the same design style when it comes to shape, size and color. This is decided by the accompanying text content. Icons standing by themselves will also follow nearby text in the same context.

Icons

Fontawesome Icons in this form. Color primary. Sizes aligned with the font size used in context.

Social Media Icons

Social Media Icons are to be placed in the footer area, right side. As the footer's color is primary, the icons are to be used in white color. Font size 3em.

Media: Image and Video

The most important thing to think about here is correct file type, low enough file size and big enough resolution.

Correct choice of content/what you see in the image is of course also important.

File size:

Image: max 200kb.

Video: Depends on content and use. Contact your team lead designer for specifics in each case. (This will be decided upon and put in this style guide.)

File type:

Image: jpg.

Video: mp4

Portrait

Min-width: 200px

Max-width: 350px

Border: none

Shadow: 0px 2px 2px -2px #666

Focus: Activity

Don't:

  • Use stock photos
  • Create effects other than box-shadow

Landscape

Min-width: 200px

Max-width: 500px

Border: none

Shadow: 0px 2px 2px -2px #666

Focus: Activity

Don't:

  • Use stock photos
  • Create effects other than box-shadow

Video

Width: 500px

Border: none

Shadow: 0px 2px 2px -2px #666

Controls for starting, stopping and volum, along with settings and the option to view expanded, are added on every video.

Don't:

  • Use autoplay!
  • Use stock videos
